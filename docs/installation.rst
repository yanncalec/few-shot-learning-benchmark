============
Installation
============


Stable release
--------------

To install Few Shot Learning Bechmark, run this command in your terminal:

.. code-block:: console

    $ pip install few_shot_learning_bechmark

This is the preferred method to install Few Shot Learning Bechmark, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Few Shot Learning Bechmark can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/yanncalec/few_shot_learning_bechmark

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/yanncalec/few_shot_learning_bechmark/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/yanncalec/few_shot_learning_bechmark
.. _tarball: https://github.com/yanncalec/few_shot_learning_bechmark/tarball/master
